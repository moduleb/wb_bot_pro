Описание

Инструмент для пользователей Wildberries, который позволяет отслеживать изменения цен на товары и получать уведомления в режиме реального времени. Бот разработан с использованием современных технологий и обеспечивает высокую производительность и надежность.
Код приложения доступен по ссылке

Возможности

  • Отслеживание цен: Принимает ссылку на товар и отправляет уведомление пользователю при изменении цены.
  • Множественное отслеживание: Позволяет добавлять несколько товаров для одновременного мониторинга.

Использование

  • Взаимодействие осуществляется через телеграм бота или оправкой запросов на API

Технологии

  1. Aiogram бот написан на этой асинхронной библиотеке
  2. FastAPI фреймворк для реализации сервера
  3. Webcokets технология связи бота и FastAPI сервера
  4. PostgreSQL SQL база данных
  5. asyncpg асинхронный драйвер для работы с PostgreSQL.
    • SQLAlchemy применяется в качестве ORM для удобного взаимодействия с базой данных.
    • Alembic используется для управления миграциями базы данных.
  8. Mkdocs: Автоматически генерируется сайт с документацией с помощью MkDocs, что облегчает понимание и использование проекта.
  9. Celery выполняет фоновые периодические задачи сравнения цен
  10. Redis селери использует его как брокера и отправляет задачи в очередь для бота
  11. gRPC операции парсинго делаются на отдельном grpc сервера
  12. Nginx обратный прокси сервер
  13. Django Админ панель
  14. docker compose Проект развернут с помощью Docker Compose, что упрощает развертывание и управление зависимостями.
